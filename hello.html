Mes commandes GIT

git pull : pour récuprer les commits

git commit : pour faire un commit

git push : poour faire envoyer un commit

git log : pour afficher les commits

git checkout[nom de la branche] : pour ce deplacer dans la branche

git checkout[main] : pour retourner dans le main(Master)

git --set-upstream origine [Nom de la branche] : pour creer la branche dans le dépôt distant

git --rebase : commande pour récuprer le commit et git fait le mergeen meme temps pour avoir une belle structure

git tag [nom] : pour creer un tag au commit

git tag -- delete : pour supprimer le tag du commit

git add . : commande a utiliser avant le commit (pour faire un save dans le fichier cacher)
